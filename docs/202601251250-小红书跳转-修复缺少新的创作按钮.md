# 小红书跳转 URL - 修复缺少"新的创作"按钮

## 问题描述
生成小红书文章后，跳转到的页面缺少"新的创作"按钮，导致无法开始新的创作流程。

## 问题原因
在 `src/background/index.ts` 中，小红书发布页面的跳转 URL 缺少 `from=tab_switch` 参数。小红书平台需要这个参数来显示"新的创作"按钮。

## 解决方案
修改了 `handlePublishToXiaohongshu` 函数中的 `publishUrl`，从：
```typescript
const publishUrl = 'https://creator.xiaohongshu.com/publish/publish';
```

改为：
```typescript
const publishUrl = 'https://creator.xiaohongshu.com/publish/publish?from=tab_switch&target=article';
```

**参数说明：**
- `from=tab_switch` - 确保页面显示"新的创作"按钮
- `target=article` - 指定打开长文章编辑器（而不是图文编辑器）

## 修改文件
- `src/background/index.ts` (第1756行)

## 验证结果

- ✅ 代码无语法错误
- ✅ 跳转 URL 修改完成

## 预期效果
现在生成小红书文章后，会跳转到包含"新的创作"按钮的正确页面，用户可以顺利开始创作流程。

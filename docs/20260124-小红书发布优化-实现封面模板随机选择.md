# 小红书封面模板随机选择功能实现 (2026-01-24)

## 功能描述
在小红书发布流程中，用户希望在众多的封面模板中随机选择一个，而不是每次都固定选择第一个。

## 实现细节
- **[xiaohongshu.ts](file:///c:/Users/ralph/IdeaProject/Memoraid/src/content/xiaohongshu.ts)**:
    - 修改了 `selectTemplateCover` 函数。
    - 使用 `Math.random()` 在所有可见的模板封面图片（`visibleImages`）中计算一个随机索引。
    - 根据随机索引选择目标图片并触发点击。
    - 增加了日志输出，显示找到的模板总数以及随机选中的序号。

## 验证结果
- 执行 `npm run build` 编译成功。
- 代码逻辑已更新，现在发布流程会自动随机挑选封面模板。

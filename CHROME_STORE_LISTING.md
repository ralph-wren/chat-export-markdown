Memoraid 是一款强大的 AI 内容处理工具，帮助您高效地总结网页内容、生成自媒体文章，并一键发布到头条号、知乎专栏和微信公众号。

• 让 AI 按照您想要的风格写作

📤 一键多平台发布
• 支持头条号、知乎专栏、微信公众号
• 标题、正文自动填入创作平台
• 智能配图：自动从热点图库搜索并插入配图
• 自动设置文章封面

💾 数据管理
• 历史记录自动保存
• 支持 Google/GitHub 登录云端同步
• 一键推送到 GitHub 仓库

📝 导出功能
• Markdown 实时预览
• 自动渲染 Mermaid 流程图
• 支持复制和下载 .md 文件

🔒 隐私保护
• API Key 仅存储在本地浏览器
• 云端同步数据端到端加密
• 不收集任何用户数据

适用场景：
• 技术人员整理学习笔记
• 自媒体作者快速创作内容
• 研究人员总结文献资料
• 任何需要高效处理网页内容的用户

立即安装，让 AI 成为您的内容创作助手！

## 权限说明（Chrome 审核必填）

### storage 权限
```
用于在用户本地浏览器中保存插件配置（如 API Key、模型设置）以及生成的摘要历史记录，确保数据不丢失且仅存储在本地。
```

### activeTab 权限
```
仅当用户点击插件图标或"摘要"按钮时，才需要临时访问当前标签页（即 ChatGPT 或 Gemini 页面）的权限，以便读取页面上的对话内容进行处理。
```

### notifications 权限
```
由于 AI 摘要生成可能需要较长时间，需要使用通知权限在任务完成或发生错误时向用户发送系统提示，即使用户切换到了其他标签页也能及时获知结果。
```

### cookies 权限
```
用于获取头条号（mp.toutiao.com）、知乎专栏（zhuanlan.zhihu.com）和微信公众号（mp.weixin.qq.com）的登录状态 Cookie，以便实现自动登录和内容发布功能。仅访问特定域名下的 Cookie。
```

### identity 权限
```
用于实现 Google 和 GitHub OAuth 登录功能，让用户可以将设置加密同步到云端，方便在多台设备间使用。
```

### host_permissions (主机权限)
```
我们的插件通过 Content Script 运行在以下网站上：
• https://chatgpt.com/* - 提取 ChatGPT 对话内容
• https://gemini.google.com/* - 提取 Gemini 对话内容  
• https://chat.deepseek.com/* - 提取 DeepSeek 对话内容
• https://mp.toutiao.com/* - 头条号创作平台自动化
• https://zhuanlan.zhihu.com/* - 知乎专栏自动化
• https://mp.weixin.qq.com/* - 微信公众号自动化

这是实现核心功能所必需的：我们需要在这些特定的 AI 聊天页面上注入脚本，以解析 DOM 结构并提取用户希望摘要的对话内容。我们只请求了这几个特定域名的权限，没有请求广泛的主机权限。
```

## 隐私政策

### 隐私政策 URL
```
https://memoraid-backend.iuyuger.workers.dev/privacy
```

### 隐私政策要点
1. 本扩展不收集任何个人身份信息
2. API Key 仅存储在用户本地浏览器的 chrome.storage 中
3. 云端同步功能使用用户自定义密钥进行端到端加密
4. 所有 API 调用直接从用户浏览器发送到 AI 服务提供商
5. 不使用任何分析或追踪工具

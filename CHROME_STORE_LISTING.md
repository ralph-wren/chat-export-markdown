## 权限说明（Chrome 审核必填）

### storage 权限
```
用于在用户本地浏览器中保存插件配置（如 API Key、模型设置、文章风格偏好）以及生成的摘要历史记录，确保数据不丢失且仅存储在本地。
```

### activeTab 权限
```
仅当用户点击插件图标或"摘要"按钮时，才需要临时访问当前标签页的权限，以便读取页面上的内容进行处理。此权限仅在用户主动触发时生效。
```

### notifications 权限
```
由于 AI 摘要生成可能需要较长时间，需要使用通知权限在任务完成或发生错误时向用户发送系统提示，即使用户切换到了其他标签页也能及时获知结果。
```

### cookies 权限
```
用于获取以下平台的登录状态 Cookie，以便实现自动登录和内容发布功能：
• 头条号（mp.toutiao.com）
• 知乎专栏（zhuanlan.zhihu.com）
• 微信公众号（mp.weixin.qq.com）

仅访问上述特定域名下的 Cookie，不会访问其他网站的 Cookie。
```

### identity 权限
```
用于实现 Google 和 GitHub OAuth 登录功能，让用户可以将设置和历史记录加密同步到云端，方便在多台设备间使用。
```

### host_permissions - <all_urls> (主机权限)
```
本扩展需要 <all_urls> 主机权限，原因如下：

1. 任意网页内容提取（核心功能）
   用户可以在任何网页上使用本扩展提取内容并生成摘要/文章。这是本扩展的核心功能，需要能够读取用户当前浏览的任意网页的 DOM 内容。

2. 链接内容获取
   在提取网页内容时，扩展会自动获取页面内相关链接的内容，以提供更完整的上下文信息。这需要能够 fetch 任意 URL。

3. 图片 OCR 识别
   扩展支持识别网页图片中的文字，需要能够获取任意来源的图片进行处理。

4. 自媒体平台发布
   支持发布到头条号、知乎专栏、微信公众号等平台，需要在这些平台页面上运行内容脚本。

我们承诺：
• 仅在用户主动触发时访问页面内容
• 不会在后台自动收集任何数据
• 所有数据处理都在本地完成，不会上传到我们的服务器
• 用户可以随时在设置中禁用特定功能
```

## 隐私政策

### 隐私政策 URL
```
http://memoraid.dpdns.org/privacy
```

### 隐私政策要点
1. 本扩展不收集任何个人身份信息
2. API Key 仅存储在用户本地浏览器的 chrome.storage 中
3. 云端同步功能使用用户自定义密钥进行端到端加密
4. 所有 AI API 调用直接从用户浏览器发送到 AI 服务提供商（如 OpenAI、DeepSeek）
5. 不使用任何分析或追踪工具
6. 网页内容仅在用户主动触发时读取，不会自动收集
